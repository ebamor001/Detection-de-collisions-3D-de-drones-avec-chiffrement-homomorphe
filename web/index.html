Vue d’ensemble

Problème

Deux drones doivent vérifier un risque de collision sans exposer leurs trajectoires (données sensibles).

Approche

Calculs géométriques sur données chiffrées (CKKS) + comparaisons booléennes (TFHE-style).

Résultat

On révèle uniquement un booléen final : collision / pas collision.

Cadre & Sujet

Projet avancé (S8) : concevoir un protocole et une implémentation pour tester l’intersection de trajectoires de drones tout en préservant la confidentialité des chemins.

Objectifs

  • Intersection en clair (2D) puis extension 3D.
  • CKKS pour les calculs réels (distances, projections, etc.).
  • Comparator TFHE-style pour les décisions booléennes.
  • Démo web + métriques de performance.
